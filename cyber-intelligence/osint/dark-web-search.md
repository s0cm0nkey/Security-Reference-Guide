# Dark Web Search

Searching the Dark Web and Deep Web is a valuable intelligence activity that can yield diverse sources of threat intelligence. These resources are useful for identifying early indicators of compromise, monitoring data breaches, and tracking emerging trends in cybercrime.

**Note:** Darknet and .onion sites change frequently. This list may not be current. Always verify site availability and exercise caution when accessing these resources.

<details>

<summary>Clearnet Search Engines</summary>

* [Ahmia](https://ahmia.fi/) - Active clearnet and .onion search engine for Tor hidden services. Provides both clearnet access and direct .onion access via Tor Browser.
* [Dark Search](https://darksearch.io/) - A clearnet search engine for searching the Dark Web. Actively maintained and functional.
* [Onion Land Search Engine](https://onionlandsearchengine.com/) - A search engine for the Deep Web. Official onion site available [here](http://3bbad7fauom4d6sgppalyqddsqbf5u5p56b5k5uk2zxsy3d6ey2jobad.onion/).
* [TorWhois](https://torwhois.com) - Query .onion addresses for information including last seen date, open ports, and running services. Command line usage: `whois -h torwhois.com [onion-address]`
* [TheDevilsEye](https://github.com/rlyonheart/thedevilseye) - Search darknet links (.onion domains) from the command line without connecting to the Tor network.
* [Torch](http://torchdeedp3i2jigzjdmfpn5ttjhthh5wbmda2rr3jvqjg5p77c54dqd.onion/) - One of the oldest and largest .onion search engines. Requires Tor Browser to access.

</details>

<summary><a href="https://ipfs.io/">IPFS - Interplanetary File System</a></summary>

A peer-to-peer hypermedia protocol designed to preserve and grow humanity's knowledge by making the web upgradeable, resilient, and more open.

* [IPFS Search](https://ipfs-search.com/#/search) - Active search engine for discovering content on IPFS.
* [Awesome IPFS Datasets](https://awesome.ipfs.io/datasets/) - Extensive collection of datasets hosted via IPFS.

<summary>Misc Sites and Tools</summary>

* [ImmuniWeb Dark Web Exposure Test](https://www.immuniweb.com/darkweb/) - Free test for dark web exposure and phishing detection. Active service.
* [OnionSearch](https://github.com/megadose/OnionSearch) - Python3 script that scrapes URLs from multiple .onion search engines. GitHub repository available.
* [TorBot](https://github.com/DedSecInside/TorBot) - Open-source intelligence tool for searching and crawling the Dark Web. Active GitHub project.
* [OnionScan](https://github.com/s-rah/onionscan) - Tool for investigating the Dark Web by scanning .onion sites for security issues and service fingerprinting. Note: Last updated 2018, but still functional.
* [Darkdump](https://github.com/josh0xA/darkdump) - Python-based dark web OSINT tool for searching and extracting data. GitHub repository available.
* [DarkOwl Vision](https://www.darkowl.com/) - Commercial dark web intelligence platform. Requires enterprise subscription.

<details>

<summary>Deprecated or Discontinued Tools</summary>

**Note:** These tools are listed for historical reference but are no longer operational, unreliable, or not recommended for use.

* **Tor2Web (tor2web.org)** - Original project discontinued due to security concerns. The concept of .onion-to-web proxies still exists but using them compromises anonymity and security. Not recommended.
* **KILOS Darknet Search Engine** - This marketplace search engine has been seized/taken offline. No longer accessible.
* **Onion Link (onionlink.online)** - This specific URL appears to be defunct. Alternative link directories exist but frequently change.
* **IACA Darkweb Tools (iaca-darkweb-tools.com)** - This resource is no longer reliably accessible and may be discontinued.
* **Dargle (dargle.net)** - Service availability is inconsistent and unreliable. Consider using alternative aggregation platforms.
* **IPSE (ipse.io)** - IPFS search project appears to have stalled/been abandoned. Use IPFS Search instead.
* **Nekhbet I2P Links** - Very outdated directory that hasn't been maintained. Historical reference only.
* **I2P Search (i2psearch.com)** - This service's reliability is questionable. Use Ahmia I2P instead for verified I2P searching.
* **Onion Search Engine (onionsearchengine.com)** - Frequently offline and unreliable. Use Ahmia or Dark Search instead.
* **OSINT Party RSS Feed (osint.party/api/rss/fresh)** - Service status uncertain and may be discontinued.
* **DarkTracer** - Service availability has become inconsistent.
* **SOCRadar DarkMirror** - Access to this tool has become limited/restricted.
* **Flare (flare.io)** - Service has undergone changes; verify current availability and offerings.

</details>
